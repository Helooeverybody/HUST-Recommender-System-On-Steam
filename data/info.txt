Please download the source data from kaggle first.
Run the Preprocess/preprocessing.py file to generate data.

Folder format: 

[name]: steam_{review count lower limit} 
    Ex: steam_40 will have >=40 reviews for each users
|-games.csv             : filtered games data
|-users.csv             : filtered users data
|-recommendations.csv   : filtered recommendations
|-games_metadata.json   : filtered games metadata
|-mapping.json          : mapping for CF
|-user_item_data.txt    : user-item-rating data
